Building Scheduling & Management System - El Pueblo Manila

Project Name: Building Scheduling & Management System

Tech Stack Used: HTML, CSS, Bootstrap

This project is for my Mini Project 1 during my bootcamp on KodeGo for Full Stack Web Development.

This project is divided into two parts: Website and Web System. The system caters to the needs of El Pueblo Manila Condominium's residents. The objective of this system is to automate manual processes and forms to make the process more accessible and convenient to the residents.
